also includes routines to search large arrays for the location of a specified
string.

!TOPIC 2809 CommaForPeriod,OpString
!NOINDEX
type
  CommaForPeriod : Boolean = False;

If True, the Form function will return '.' for ',' and vice versa. This
makes it a bit easier to write programs that can be used both in the U.S.
and in countries that use commas for decimal points.

See also:  2743Form
