!TOPIC 554 OpDevice
OpDevice adds knowledge of specific printers to 2127OpPrint's general framework
for printing. OpDevice doesn't declare any objects. Its primary export is a
category of routines called hardware registration procedures. A hardware
registration procedure configures an instance of a Printer object for a
specific printer. OpDevice also contains various primitive routines for
printhead positioning, line drawing activation and the like, but you won't
need to refer to them directly.

  0586Hardware registration routines  0587Device primitives  0588Printer numbers
  0589Symbol sets

!TOPIC 586 Hardware registration routines
!NOINDEX
!NOSEARCH
Here are all of the hardware registration procedures provided by
OpDevice. A hardware registration procedure customizes a printer instance
to include specific option flags, PrintModes, character translation tables,
a default font, and procedure pointers for printhead positioning, and so
on. Hardware registration procedures do not affect the underlying
2212BasePrinter, the X or Y ratios, or the user-written printer error
handler.

    0566DiabloRegister                    0567EpsonFXRegister

!TOPIC 589 Symbol sets
!NOINDEX
!NOSEARCH
const
const
  {LJ Symbol Sets}
  ljssUSAscii      = '0U';   {normal US Ascii}
  ljssIBMUS        = '10U';  {special Ascii + IBM extended (avail. on LJII)}
  ljssUSLegal      = '1U';   {US Legal}
  ljssRoman        = '0E';   {Roman extensions}

  SymbolSet        : String[4] = ljssUSAscii;

SymbolSet is the symbol set used to select fonts on LaserJet and compatible
printers.  The ljssXXX constants are a few of the more common symbol sets,
but SymbolSet may assume any valid symbol set value.
