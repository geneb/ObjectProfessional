!TOPIC 1129 OpEnhKbd
OpEnhKbd invisibly installs a pair of interrupt handlers which trap all
system calls that request keyboard input. Besides activating or emulating the
enhanced keyboard, OpEnhKbd makes the Alt key act like a normal shift key
where the numeric keypad is concerned. Hence, Alt-Left, Alt-Up, and the
rest can be used as additional keystrokes for an application. Note, however,
that this disables the normal ability to enter arbitrary ASCII characters
using the Alt and the keypad. This capability is still possible by pressing
LeftShift and Alt simultaneously.

  1133InitKbdVectors             1134RestoreKbdVectors        1132FiltersEnhanced
;
;---------------------------------------------------------------------------
!TOPIC 1130 EnableEnhanced
!NOINDEX
const
  EnableEnhanced : Boolean = True;

Set False to temporarily disable the enhanced keyboard interrupt handlers.

See also: 1134RestoreKbdVectors
;
;---------------------------------------------------------------------------
!TOPIC 1131 HasEnhancedKbd
!NOINDEX
var
  HasEnhancedKbd : Boolean;

True when a real enhanced keyboard is detected.
;
;---------------------------------------------------------------------------
!TOPIC 1132 FiltersEnhanced
!NOINDEX
var
  FiltersEnhanced : Boolean;

This variable is set to True by OpEnhKbd if it has been determined that
the current machine's BIOS "filters" key codes whose scanword is greater
than $8500.
;
;---------------------------------------------------------------------------
!TOPIC 1133 InitKbdVectors
!NOINDEX
procedure 1129OpEnhKbd.InitKbdVectors

Save and setup vectors for interrupts $09 and $16.  This routine should be
called only if 1134RestoreKbdVectors has been called first.
;
;---------------------------------------------------------------------------
!TOPIC 1134 RestoreKbdVectors
!NOINDEX
procedure 1129OpEnhKbd.RestoreKbdVectors;

Restores original vectors for interrupts $09 and $16. Needed only when using
OpEnhKbd in a TSR.

See also: 1130EnableEnhanced
;
